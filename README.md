# Check 'em Up

## How to Use
Clone the repository to your directory.

Use
```Python3
python main.py
```
to start the game.

## Keyboard Configuration
Player 1:\
T,G,F,H - move\
x - select a square / shoot a bullet\
z, c - change view in shooter\
\
Player 2:\
UP,DOWN,LEFT,RIGHT - move\
. - select a square / shoot a bullet\
,, / - change view in shooter
